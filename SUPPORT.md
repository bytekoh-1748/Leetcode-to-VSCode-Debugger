# Support

If you run into a bug, have a feature request, or want to suggest an improvement:

- Open an issue at <https://github.com/bytekoh-1748/Leetcode-to-VSCode-Debugger/issues>
- Include your VS Code version, Python version, and a sample solution and case file when possible

For publishing or maintenance questions, start a GitHub issue as well so the discussion stays attached to the project history.
